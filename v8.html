V8

Qu'est-ce que V8

V8 est un moteur JavaScript(utilisé par Node.js, Chrome et d'autre) permettant de convertir directement votre code JavaScript en langage machine avant d'éxécuter celui-ci, ce qui résulte en une optimisation maximale lors de l'éxécution du code.

Date Changements

2021
Septembre

Version : 9.4

Ajouts : - La classe d'initialisation 'static { }' qui permet d'initialiser du code à éxécuter une seule fois.
- Il y a aussi eu des changements au niveau de l&39;API v8 mais je n'ai pas réussi à comprendre les changements opérés.

La première image de septembre montre la classe "static { }" en action.

Version : 9.5

Ajouts : - Lors de la version 8.1 de V8, l'API Intl.DisplayNames avait été ajouté, dans cette nouvelle version, cette API à été améliorée rajoutant deux nouveaux types : calendar et dateTimeField(vous pouvez les voir dans la deuxièmeimage d'exemple.
-Une amélioration du type 'language' a été fournie où ce type supporte les dialectes(comme l'anglais Australien, américain, anglais, ...).
Octobre

Version : 9.6

Ajouts : - La version 9.6 ajoute un support pour type de variable interne à WebAssembly : anyref ou externref(le vrai nom).
Cette nouvelle variable interne permet de faire le lien plus facilement entre les objets JavaScript(fonctions, classes) et le module WebAssembly. - Pour plus d'informations à propos de cette variable, je vous recommande ces explications(en anglais), elles m' permises de mieux comprendre la variable externref
Le lien : https://rustwasm.github.io/wasm-bindgen/reference/reference-types.html
Novembre

Version : 9.7

Ajouts : - De nouvelles méthodes ont été ajoutées comme la méthode "findLast" et la méthode "findLastIndex".
Ces méthodes permettent de rechercher le dernier index ou le dernier &eacurte;lément qui correspond avec le test fourni.
Décembre
